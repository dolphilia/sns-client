> テキストベースのラフモックアップ。構成・レイアウト・要素の配置を確認するためのもの。細かい色やデザインの詳細はプロトタイプ実装時に決める。

## 共通レイアウト（ログイン後）

```
┌─────────────────────────────────────────────────────┐
│  ┌──────────┐  ┌───────────────────────────────┐    │
│  │          │  │                               │    │
│  │  サイド  │  │       メインコンテンツ        │    │
│  │  バー    │  │                               │    │
│  │  240px   │  │        640px（固定幅）        │    │
│  │          │  │                               │    │
│  └──────────┘  └───────────────────────────────┘    │
└─────────────────────────────────────────────────────┘
```

**設計の意図**
- 右側の余白は意図的に空けておく。情報密度を下げ、視覚的な「余裕」を作る
- メインカラムは最大 640px に固定。画面幅が広くても情報が横に広がらない
- サイドバーはナビゲーションのみ。ウィジェット・トレンド・おすすめは置かない

## サイドバー

```
┌────────────────────┐
│                    │
│  ◎ アプリ名       │  ← ロゴ/名称
│                    │
│  ─────────────     │
│                    │
│  ⌂  ホーム        │  ← 現在地をハイライト
│  🔔 通知          │  ← バッジなし（意図的）
│  👤 プロフィール  │
│                    │
│  ─────────────     │
│                    │
│  ⚙ 設定          │
│                    │
│  ─────────────     │
│                    │
│  [アバター]        │
│  表示名            │  ← 下部に固定
│  @handle          │
│                    │
└────────────────────┘
```

**設計の意図**
- 通知アイコンに**バッジ（未読数）を表示しない**。「確認しなければ」というプレッシャーを排除する
- トレンド・おすすめユーザーのウィジェットは置かない。比較・FOMO の誘発を防ぐ

**投稿カードの詳細**
```
┌──────────────────────────────────────────────┐
│ [avtr]  表示名          @handle  ・  3時間前 │
│                                              │
│  投稿テキストがここに入る。                  │
│  メンション @user や URL も表示される。      │
│                                              │
│  ──────────────────────────────────────────  │
│  💬        🔁        ♡        ↗            │
│  返信      RT        いいね    共有          │
└──────────────────────────────────────────────┘
```

**設計の意図**
- **いいね数・リポスト数・返信数はすべて非表示**。アイコンのみ。数字を見せないことで他者との比較を起こさない
- タイムスタンプは「3時間前」などの相対表示（「13:42」のような絶対時刻は使わない）
- 「もっと読む」ボタンで区切る。無限スクロールにしない
- 投稿フォームはタイムライン上部に常時表示（使いやすさ優先）

## 色・タイポグラフィの方針（大まかな方向性）

- **基調**: 白 / 薄いグレー系（明るく清潔感のある雰囲気）
- **アクセント**: 控えめな青（Bluesky のブランドカラーに近い系統）
- **フォント**: システムフォント優先（OS ネイティブな読みやすさ）
- **文字サイズ**: 投稿本文は 15px 以上。詰め込みすぎない行間（1.6 前後）
- **カード間の余白**: タイトすぎない余白。1 投稿ごとに視覚的な区切りを明確にする
- **ダークモード**: 実装時に対応（Tailwind の `dark:` クラスで管理）
